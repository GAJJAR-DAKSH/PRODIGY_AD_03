# ⏱️ Stopwatch App

A simple Android Stopwatch application developed using Java and Android Studio.

## 🚀 Features

* Start Stopwatch
* Pause Stopwatch
* Reset Stopwatch
* Display Minutes, Seconds, and Milliseconds
* Record Lap Times
* View Lap Time History
* User-Friendly Interface

## 🛠️ Technologies Used

* Java
* Android Studio
* XML
* Handler
